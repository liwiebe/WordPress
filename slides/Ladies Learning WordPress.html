What is WordPress?

An Open Source Content Management System used to power thousands of websites

Initially was used to power blogs

Powerful enough to run large websites and applications

Used by TechCrunch, CNN, Ladies Learning Code and many more...

WordPress.com vs. WordPress.org

WordPress.com is hosted for you - no servers or code required!

It doesn't allow for a lot of customization, though. No plugins or custom themes.

We will be using WordPress.org

This allows for 100% customization - themes and plugins oh my!

Code + Database = ♥

The database holds our data and the code defines how and when the data is printed to the page

The two work together to output HTML

/wp-admin and /wp-includes holds the engine of WordPress - we don't touch it.

/wp-content holds our specific content like themes, plugins and image uploads.

What you need to have installed or downloaded:

An application that can run your local server (MAMP, WAMP or XAMPP)

A text editor like Sublime Text 2

Your WordPress ZIP file from WordPress.org

Localhost

Local Server (MAMP, WAMP or XAMPP): A small service that runs on your laptop which mimics the functionality of a large, paid server.

We are able to work on our website without uploading it anywhere.

Text Editor: A program that runs on your computer, used for editing code

Code is written in plain text, and we use a text editor instead of NotePad or TextEdit because it makes things a lot easier on you

I recommend Sublime Text 2 (unlimited trial) but other options like NotePad++, TextWrangler and TextMate work too. Your mentors might have their own favourites

Installing WordPress!

Installing WordPress can be a little tricky the first time, so pay attention here and it will all go smoothly!

Step 1: Unzip the WordPress package you downloaded and you'll have a folder called WordPress

Step 2: Move that folder over to your local server (localhost)

TIP: Make sure you only have one folder called WordPress. You shouldn't have a folder called WordPress inside of one called WordPress!

If you need help finding the folder: Windows: Click WAMP icon in tray → www directory Mac: click MAMP icon → Preferences → Apache Tab

Step 3: Open up PHPmyadmin and create a database called llc

Windows: Go to http://localhost Mac: Go to http://localhost:8888/MAMP/ Both: click PHPMyAdmin tab

Step 4: Rename wp-config-sample.php to wp-config.php

Step 5: Open wp-config.php in your text editor

Step 6: Change these settings:

Windows Users: Leave password blank.

Step 6: Surf to http://localhost/wordpress or http://localhost:8888/wordpress in Chrome (or your favorite browser like Firefox or IE10) and you're ready to install!

PHP

You've probably noticed that a lot of files in your WordPress folders end with ".php"

PHP is the programming language WordPress is written in

PHP tags: When we need to use PHP, we put it inside <?php and ?> tags so our server will know to process it as PHP

Let's look at our wp-content files

Different pages can be displayed with different layouts

Example: your home page (index.php) might look different than your category page (category.php)

Let's take a look at some of the more important template files in our theme

index.php - The front page of your blog

single.php - Default template used for displaying a single post

category.php - Displays all posts within a certain category

footer.php - The footer that is appended at the bottom of every page

header.php - The header that is appended to the top of every page

sidebar.php - Sidebar that is appended to every page

loop.php - A reusable file to display posts (much more on this later!)

Anatomy of a WordPress Theme

A WordPress theme is made up of a number of these .php files (template files) that are used to display different pages on your site

For Example...

single.php is used to show a single blog post while index.php is used to display the main blog page listing your most recent posts. More on this later.

Theme Layout Basics

Most template pages can be stripped down the following:

<?php get_header(); ?>
        { The Page's Content and the Loop live Here}
        <?php get_sidebar(); ?>
        <?php get_footer(); ?>

Introduction to The Loop

The loop is the lifeblood of a WordPress theme.

Here's an example: on your blog page, it's used to display your most recent blog posts over and over - until it hits that limit of 5 blog posts per page that we set earlier.

So our blog page loop shows 5 posts, but a single post (single.php) only shows 1 post.

So what?

The Loop is what makes WordPress work

On every page, you have opening tags, title, content, closing tags. Nothing on the HTML side ever changes

Then the loop pulls out the content that you plugged into your WordPress dashboard and displays it

This is part of what lets you go into your dashboard and add posts and pages without touching the design (theme) of your site

We need to know how the loop works so that we can edit it.

The Loop Steps

1. The blog page is requested by the user. Wordpress asks the database for all content for that page

2. The database returns 5 of the latest blog posts

3. Wordpress first grab the header.php file and starts the page output

4. Wordpress then hits the loop. Since we have 5 posts from the database, we output the code within the loop 5 times

5. Wordpress is done with the loop, so it moves on and outputs the sidebar and the footer

The Code

A stripped down look outputting only the title and the content.

<?php if ( have_posts() ) while ( have_posts() ) : the_post(); ?>

          <h2><?php the_title(); ?></h2>
          <?php the_content(); ?>

          <?php endwhile; ?>

Still Confused? Don't Worry!

Our WordPress theme comes with the loop.php which handles most of this for you

You just need to be aware of how the loop works and what it looks like so you can edit it!

Template Tags

Wondering how to actually get your content onto the page while you're inside the loop?

Template tags are short little PHP functions that will output content from your site

No knowledge of PHP is required

Most template tags only work within the loop

CSS 101

Before we can start customizing the look of our WordPress theme, we need to learn the basics of CSS. We won't be diving too deep into CSS, as it's a huge topic on its own, but let's take a quick look at it

CSS stands for Cascading Style Sheets, they allow you easily add style to the elements on your website. Examples of elements are heading tags, images, divs (content and sidebars), links and lists

A basic CSS definition has three parts: selector, property and value

Selectors

The Selector defines which element the CSS will be applied to.

<div id="wrapper"> ... </div> 
          div#wrapper { ... }

        <div class="post"> ... </div> 
          div.post { ... }

        <span class="date"> ... </span> 
          span.date { ... }

        <span class="category"></span> <div class="meta"></div>  
          span.category, div.meta { … } 

        <div class="parent"><div class="child"></div></div> 
          div.parent div.child { ... }

The Property and Value

The property defines which style is being set on the current selector and the value sets it.

There are hundreds of CSS properties. Here a few popular ones for a div with the ID of wrapper

#wrapper {
          background : black;
          border:2px solid green;
          font-family:Georgia, sans-serif; 
          }

There are a bunch of them on your cheat sheet. Let's go through them now...

a {
          text-decoration: none; /* Removes all underlines from links */
          color: blue; /* Changes all links to blue */
          }
a:hover {
          color: red; /* Changes all links to red  only when we hover*/
          }
.post {
          background : url(relative/pathp/to/the/background.png); /* Add an image to the background */
          border: 3px solid yellow; /* add a yellow border */
          padding:20px; /* Increase padding to 20px */
          }
.post h2 a { /* finds .post -> Heading 2 -> links */
          font-size:40px; 
          font-family: georgia, sans-serif; /* change the font */
           border-bottom:3px solid goldenRod; /* Changes the bottom border only to goldenRod colour*/
          }
ol li { /* Selects all ordered list items */
          list-style-type: upper-roman;  /* Changes from 1,2,3 to I, II, III...*/
          }
list-style-type: square; /* changes from circle bullets to squares*/
         font-style:italic;
        }
.more {
        float: left/right/none; /* floats an element to the left or right of its relative parent */
        text-align: center/left/right/inherit;
        margin: 10px 20px 15px 25px; /* Top Right Bottom Left margin values*/
        font-weight:bold;
        width:100%;
       height:500px;
       overflow:scroll; /* anything over 500px high will have scrollbars */
      }

Step 5: Right underneath <?php endwhile; ?>, create an empty div with the class of homeColumn.

Step 7: Create another column with the class of homeColumn but this time inside we're going to put the template tags for pulling in the latest posts
